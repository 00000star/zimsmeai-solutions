Triggered when a client pays the deposit for a Launch Ready package.

## What LaunchCoach Must Do

1. Read the client details attached to this task
2. Run `homework-collector` in business-launch mode
3. Produce:
   - business name shortlist
   - brand direction brief
   - 1-page business plan
   - pricing strategy
   - 12-month projection
   - first 30 days plan
4. Create a linked delivery request for WebBot for the included Basic website
5. Post all deliverables or links in the task
6. Mark the task `in_review` and hand back to FrontDesk
