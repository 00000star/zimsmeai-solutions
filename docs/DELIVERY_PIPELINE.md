# Delivery Pipeline

## Website Jobs

1. lead enters via WhatsApp
2. FrontDesk qualifies and confirms package
3. deposit is confirmed
4. client homework is collected
5. ProjectOps verifies completeness
6. WebBot builds from the correct template family
7. QualityLead checks the output
8. FrontDesk sends staging or ZIP handoff
9. revisions are handled
10. final ZIP is delivered
11. follow-up and testimonial request

## AI Jobs

1. identify one practical workflow first
2. define trigger, action, and fallback
3. build the simplest reliable version
4. QA test the flow
5. deliver instructions and support offer

## App Jobs

1. run discovery first
2. define user roles and top 3 flows
3. decide prototype, PWA, or MVP
4. quote only after scope is clearer
