# Web Presence Factory

This is the primary revenue engine.

The point of this project is not to make every site from scratch. The point is to combine structured WhatsApp intake, real client assets, reusable layouts, and disciplined QA into a premium website delivery machine.

## Core Pipeline

Lead → package selection → deposit → homework intake → ProjectOps verification → WebBot build → QualityLead QA → client review → final ZIP handoff → follow-up

## Notes

- Work does not start until the required homework is complete.
- Every build should target a clean deployable ZIP.
- Every job passes through QA before client handoff.
- FrontDesk owns revisions and upsells.
