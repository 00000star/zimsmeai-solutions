# Mobile App Development

This project should be treated carefully.

The goal is not to sell every enquiry as a full mobile app. The goal is to help the client choose the right level of product:

- prototype
- installable PWA
- internal tool MVP
- broader custom app only after discovery

## Delivery Model

| Phase | Output | Owner |
|---|---|---|
| Discovery | scope, user flows, risk notes | AppMaster |
| Prototype | clickable concept or wireframe set | AppMaster |
| Recommendation | PWA vs MVP vs larger build | AppMaster + CTO |
| Build | only after approved scope | AppMaster |
| QA | test pass and release notes | CTO + QualityLead |
| Handover | credentials, notes, next-step plan | FrontDesk |
