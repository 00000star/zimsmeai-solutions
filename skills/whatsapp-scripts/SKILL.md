# WhatsApp Scripts

Use these scripts as structured starting points. Personalise the first sentence whenever possible.

## 1. First Enquiry Reply

```text
Hi [Name]! 👋 Thanks for reaching out to ZimSME AI.

I checked out [business name / page] and noticed [specific observation].

We help businesses like yours with [website / chatbot / app / consulting outcome].

Quick question: are you looking to fix this now, or are you still exploring options?
```

## 2. Deposit Request

```text
Perfect — based on what you've told me, the best fit is our [package] at $[price].

To secure your slot, we take 50% upfront and 50% when the work is ready.

EcoCash: [number]
Innbucks: [number]

Once you've sent it, please send the screenshot here and I'll open your project immediately.
```

## 3. Homework Reminder

```text
Hi [Name] — just checking in on the items we need so we can continue:

- logo
- business description
- services/products
- photos
- contact details

The sooner you send them, the sooner we move your project forward. 🙏
```

## 4. Staging Link Delivery

```text
Hi [Name]! Your draft is ready ✅

Here's your staging link:
[link]

Please review it and send any changes in one message so we can handle them efficiently.
```

## 5. Balance Request

```text
Great news — your project is ready to go live ✅

The remaining balance is $[amount].

Once payment is confirmed, we'll complete handover / publish the site / release the final files.
```

## 6. Referral Ask

```text
I'm glad you're happy with the result 🙌

If you know another business owner who needs help with their website, AI setup, or launch support, please refer them to us. Referrals really help us grow.
```

## 7. Coaching / Consulting Follow-Up

```text
Hi [Name] — following up on your report and next steps.

The fastest win from what we found is:
[one action]

Would you like us to help you implement it, or would you prefer to continue with [next package]?
```
