Image placeholders are included so template previews do not break.

Replace these before client delivery:
- `logo.png` → real client logo
- `hero.jpg` → hero/banner image
- `about.jpg` → about/team/service image
- `gallery-*.jpg` / `portfolio-*.jpg` → real work or approved visuals
- `team-*.jpg` → real team photos or approved portraits

Keep filenames the same if you want to avoid editing the HTML.
