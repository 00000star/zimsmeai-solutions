# Corporate Template

This folder contains the multi-page corporate site starter.

## Included pages

- index.html
- about.html
- services.html
- portfolio.html
- blog.html
- blog-post-1.html
- blog-post-2.html
- blog-post-3.html
- faq.html
- team.html
- contact.html
- testimonials.html
- privacy.html
- terms.html
- 404.html

## Shared assets

- `styles.css` — shared styling for all pages
- `script.js` — mobile navigation behaviour
- `nav.html` — reusable header snippet for agent reference
- `images/` — starter placeholders and asset instructions

## Important

Replace placeholders before delivery:
- text placeholders like `{{BUSINESS_NAME}}`
- legal boilerplate
- sample posts and testimonials
- image placeholders
